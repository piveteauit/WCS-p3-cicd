Ajouter un webhook sur Github

A propos des Webhooks

Partie 1

  1. 1 - Depuis le repo, aller dans Settings
  2. 2 - Ensuite aller dans Webhooks
  3. 3 - Cliquer ensuite sur Ajouter un webhook

Partie 2

  1. 1 - Pour le payload url, utiliser: https://btz-dev.fr/cicd/
  2. 2 - Pour le content-type, chosir application/json
  3. 3 - Conserver la verification SSL
  4. 4 - Pour les events, utiliser seulement les evenements push
  5. 5 - Bien verifier que la checkbox "Active" est on
  6. 6 - Valider le formulaire, et c'est tout bon
